## 更新内容

- 接入 Sparkle 2.9.2，macOS APP 现在可以在应用内自动检查、下载并安装后续正式版本。
- 更新包与更新清单均使用 EdDSA 签名校验，安装时采用安全的原子替换并自动重新打开应用。
- “帮助 → 检查更新...”现在直接打开 macOS 原生更新窗口，不再跳转 GitHub 下载页面。
- 完善 macOS 构建与发布流程：固定并校验 Sparkle 依赖、验证成品可启动，并自动生成签名更新清单。
- 源码运行、Windows 包及缺少 Sparkle 的异常 macOS 包仍保留 GitHub Release 手动更新后备方案。

## 从旧版本升级

`v1.1.0` 及更早版本尚未包含自动更新组件，需要手动下载并安装本次 `v1.2.0`。安装后，后续版本即可直接在应用内自动更新。

## 系统要求

- Apple Silicon Mac
- macOS 26.0 或更高版本

本次更新未修改任何图像处理与调校算法。
