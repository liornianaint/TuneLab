## 更新内容

- ColorChecker 图片输入默认改为“标准色卡逼近”：直接使用标准或自定义目标的 24 个 sRGB 色值，等权最小化平均 CIEDE2000 ΔE00。
- 图片模式不再以单色块回退、Pass Rate、重点 Patch、Strategy、Regularization、强度或饱和度作为候选门禁；局部回退会明确提示，但不会否决平均色差更低的结果。
- 继续保留系数范围、Neutral 轴、可逆性和 Matrix Health 约束，避免生成数值失控或无法写入 Qualcomm XML 的矩阵。
- 改后模拟图始终使用本轮实际 Delta CCM，与 XML 写回结果保持一致；移除旧的实拍 Profile 和独立完整还原预览路径。
- CSV 输入继续使用原有的 ΔE/ΔC/Δh、Pass Rate、重点 Patch 与 Regression Protection 工程优化，不受本次策略调整影响。
- 使用工程内 8 张 ColorChecker 测试图回归，24 色平均 ΔE00 均下降；其中 `A.jpg` 从 8.515 降至 7.188。

## 更新方式

`v1.2.0` 用户可由应用内 Sparkle 更新器自动下载并安装本版本。`v1.1.0` 及更早版本需要手动下载新版 macOS APP。

## 系统要求

- Apple Silicon Mac
- macOS 26.0 或更高版本
